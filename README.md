# 嘉義市國中藝才班術科測驗 — 線上練習

收錄 111–114 學年度嘉義市國民中學藝術才能班術科測驗：
- 樂理及音樂基本常識
- 聽音測驗（使用 Salamander Grand Piano 真實鋼琴錄音）

## 線上使用

直接造訪 GitHub Pages 部署的網址即可作答；按「交卷檢查」後每題下方會顯示官方正解。

## 內容

- `index.html` — 入口頁
- `exam.html?id=<year>-theory` — 樂理測驗（111–114 學年度）
- `listening.html?id=<year>-listening` — 聽音測驗（112–114 學年度；111 因原試卷未公布譜面，本站不提供）
- `*.pdf` — 8 份原試卷／答案 PDF
- `images/questions/` — 各題譜例（從原 PDF 抽取）

## 來源

題目來源：嘉義市國民中學藝術才能班術科測驗 111–114 學年度公開試卷／答案。
